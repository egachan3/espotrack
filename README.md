# espotrack

League of Legends・Valorantなどの試合・トーナメント情報を
一覧できるeスポーツダッシュボード。

## Data source

Data provided by [Liquipedia](https://liquipedia.net) under 
[CC-BY-SA 3.0](https://creativecommons.org/licenses/by-sa/3.0/).

## Setup

1. Clone the repo
2. Copy `.env.example` to `.env.local` and fill in your keys
3. `npm install`
4. `npm run dev`

## API Keys

- Liquipedia API: https://liquipedia.net/api
- Twitch API: https://dev.twitch.tv/console
